 -Movie-Recommendation-
The Movie Recommendation System is a machine learning-based application designed to suggest movies to users based on their preferences. It uses a content based filtering approach  where recommendations are generated by analyzing similarities between movies.
Movie Recommendation System

Hey! This is a simple Movie Recommendation System I built using Python and basic machine learning concepts. The goal of this project is to suggest movies based on what a user likes.

💡 What this project does
You enter a movie name 🎥
The system finds similar movies
It recommends a few movies you might enjoy

It works by comparing movie genres (and optionally keywords) and finding similarities between them.

⚙️ How it works (in simple words)
The movie data is stored in a CSV file
Text data like genres is converted into numbers using a technique called CountVectorizer
Then, cosine similarity is used to compare movies
Based on similarity scores, recommendations are generated
🛠️ Tech used
Python
Pandas
Scikit-learn
▶️ How to run
Make sure your Python file and movies.csv are in the same folder
Open terminal in that folder
Run:
python AIMLproject.py
Enter a movie name and get recommendations 🎬
